# Taylor Stitch Clone

A responsive, multi-page frontend clone of the Taylor Stitch website, featuring modern UI patterns and e-commerce layouts.

## 🌟 Overview
This project faithfully replicates the look and feel of the Taylor Stitch e-commerce platform. The project is completely responsive across all devices (Desktop, Tablet, and Mobile) with interactive elements such as a functional mobile hamburger menu.

## ✨ Key Features
- **Fully Responsive Design**: Optimized for 1200px, 992px, 768px, and 576px breakpoints to guarantee an excellent mobile, tablet, and desktop experience.
- **Working Elements**: Integrated Bootstrap 5 Javascript bundle enables a fully functioning hamburger toggle menu for mobile views.
- **Multi-Page Layout**:
  - **`index.html`**: The stylized landing page highlighting categories, newest arrivals, house shirts, and dispatch articles.
  - **`secondPage.html`**: A dedicated product listing page with a sidebar and product grids.

## 🛠️ Built With
- **HTML5** for semantic page structure.
- **Vanilla CSS3** for accurate and custom styling over multiple breakpoints.
- **Bootstrap 5.3** for navigational components.

## 🚀 Getting Started
1. Clone the repository to your local machine.
2. Open `index.html` in your web browser to view the Taylor Stitch landing page.
3. Use the navigation links to jump over to the Shirts & Sweaters (`secondPage.html`) section!
